记一次大数据模型开发

前言：与你所见，这是一次大数据模型开发，也是我的想死之路，也算入门了一下ai了

基本步骤

大数据预测模型制造的基本步骤包括以下几个阶段：

  1. 制订目标：明确预测模型的目标，即你希望模型能够解决什么问题。
  2. 数据收集：收集与预测目标相关的数据。这些数据可以来自各种来源，例如数据库、文件、API等。
  3. 数据预处理：对收集到的数据进行清洗和格式化，以便于模型可以接受。这可能包括处理缺失值、异常值，进行数据转换等。
  4. 特征工程：选择和构造对预测目标有意义的特征。这可能涉及到特征选择、特征转换、特征提取等步骤。
  5. 模型训练：选择合适的机器学习算法，并使用收集到的数据对算法进行训练。训练过程中，模型会学习到数据中的规律和趋势。
  6. 模型评估：使用一部分未参与训练的数据来评估模型的性能。这可以帮助我们了解模型在未知数据上的表现。
  7. 模型优化：根据模型评估的结果，对模型进行调整和优化，以提高其预测性能。
  8. 模型部署：将训练好的模型部署到生产环境中，用于实际的预测任务。

run()：训练模型的函数，主要用于训练一个用于序列分类的模型。

  1. paddle.set_device('cpu')：设置PaddlePaddle运行的设备为CPU。
  2. model = AutoModelForSequenceClassification.from_pretrained(self.model_name, num_classes=self.num_classes)：从预训练模型中加载一个用于序列分类的模型。
  3. tokenizer = AutoTokenizer.from_pretrained(self.model_name)：从预训练模型中加载一个分词器。
  4. train_ds = self.dataset.getLoader(mode='train') 和 valid_ds = self.dataset.getLoader(mode='valid')：获取训练和验证数据集。
  5. trans_func = functools.partial(preprocess_function, tokenizer=tokenizer, max_length=self.max_length)：定义一个预处理函数，用于将原始数据转换为模型可以接受的格式。
  6. train_ds = train_ds.map(trans_func) 和 valid_ds = valid_ds.map(trans_func)：对训练和验证数据集进行预处理。
  7. collate_fn = DataCollatorWithPadding(tokenizer)：定义一个函数，用于在数据加载时对数据进行填充和批处理。
  8. train_data_loader = DataLoader(...) 和 valid_data_loader = DataLoader(...)：定义训练和验证的数据加载器。
  9. self.log_writer = LogWriter(self.cf.logPath)：定义一个日志写入器，用于记录训练过程中的信息。

模型评估

evaluate():用于评估模型在给定数据集上的性能。

  1. model.eval()：将模型设置为评估模式。在评估模式下，模型的某些特性（如Dropout和BatchNorm）会有所不同，以确保能够正确地评估模型的性能。
  2. total_loss = 0.0：初始化总损失为0。这个变量用于累计所有批次的损失。
  3. metric = Accuracy()：创建一个准确率度量对象。这个对象用于计算模型的准确率。
  4. for batch in data_loader:：遍历数据加载器中的每一个批次。
  5. input_ids, token_type_ids, labels = batch['input_ids'], batch['token_type_ids'], batch['labels']：从当前批次中提取输入数据和标签。
  6. logits = model(input_ids, token_type_ids)：将输入数据传入模型，得到模型的输出。
  7. loss = F.cross_entropy(logits, labels)：计算当前批次的损失。
  8. total_loss += loss.item()：将当前批次的损失加到总损失上。
  9. correct = metric.compute(logits, labels)：计算当前批次的准确率。
  10. metric.update(correct)：更新准确率度量对象。
  11. avg_loss = float(total_loss) / len(data_loader) if len(data_loader) > 0 else 0：计算平均损失。
  12. accuracy = metric.accumulate()：计算总体准确率。
  13. return avg_loss, accuracy：返回平均损失和准确率。

rank():用于在每个训练周期（epoch）结束后，对模型进行评估并保存模型参数。

  2. def save_model(state_dict, file_path):：定义一个函数，用于保存模型参数。
  3. rank_data = {...}：创建一个字典，包含了当前训练周期的相关信息，如训练损失、验证损失、训练精度和验证精度等。
  4. rank_file = self.rankPath + 'rank.csv'等：定义了一些文件路径，用于保存模型参数和训练记录。
  5. update_csv(record_file, rank_data)：更新训练记录文件。
  6. if os.path.exists(rank_file):：如果排名文件存在，则读取文件中的最高验证精度。如果当前周期的验证精度高于最高验证精度，则更新排名文件，并保存当前的模型参数。
  7. save_model(model.state_dict(), record_model_file)：无论验证精度是否提高，都保存当前周期的模型参数。
